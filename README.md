# FUTURE_PE_02
# AI Content Marketing using UGC Ads

Task 2 – Prompt Engineering Internship (Future Interns)

---

## Project Overview

This project demonstrates how AI can be used to generate high-converting marketing content using structured prompts.

The goal was to create UGC-style ad scripts and marketing content that can be used for social media platforms like Instagram Reels, YouTube Shorts, and Facebook Ads.

---

## Product Selected

Product Name: Boat Airdopes 141  
Category: Wireless Earbuds  
Target Audience: Students and young professionals  

---

## Objective

Design structured prompts that generate:

- UGC-style ad scripts
- Scroll-stopping hooks
- Problem–solution advertisements
- Social media captions
